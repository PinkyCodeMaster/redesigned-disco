# Welcome to Athena!

This guide will walk you through the steps to get started with your Athena project, from setup to running your application.

## Prerequisites

Make sure you have the following installed:

- Bun
- Node.js
- PostgreSQL (via Neon)

## Steps

1. **Clone the Repository**:
    ```bash
    git clone https://github.com/your-repo/athena.git
    ```

2. **Install Dependencies**:
    ```bash
    bun install
    ```

3. **Run the Development Server**:
    ```bash
    bun dev
    ```

4. **Database Setup**: Follow the [Database Setup Guide](../api/database-setup) to configure PostgreSQL.
