## Athena Project Overview

Welcome to Athena, an all-in-one platform for managing eCommerce and business services. This documentation will guide you through setting up and using the Athena platform, including its frontend (built with Next.js), PostgreSQL integration, and more.

### Key Features

- **Native Next.js Frontend**: Build fast and scalable applications with the latest Next.js features.
- **PostgreSQL Database Setup**: Use Neon to set up and manage your PostgreSQL database with ease.
- **Node.js Runtime**: Use the latest Node.js runtime for fast, reliable server-side execution.
- **Add-ons**: Enhance your project with Starlight, Biome, Turborepo, and Husky for additional functionalities like better developer workflows and continuous integration.
- **AI Example**: Check out the included "AI" example for integration into your Athena project.

## Getting Started

Follow these steps to get started with your Athena project:

1. **Clone the repository**:
    ```bash
    git clone https://github.com/your-repo/athena.git
    ```

2. **Install dependencies**:
    ```bash
    bun install
    ```

3. **Run the development server**:
    ```bash
    bun dev
    ```

4. **Set up your PostgreSQL database** with Neon. Instructions can be found in the database setup section of the docs.
